# BBQ

A modern baseball draft platform. BBQ = Baseball + Queue.

## Getting Started

1. To run locally you must have a Docker instance running:

```bash
pnpm run db:generate
pnpm run db:up
```

2. Install dependencies:

```bash
nvm use
pnpm install
```

3. run the development server:
```bash
pnpm dev
```

4. View [http://localhost:3000](http://localhost:3000).
