## 📐 User Interface Concepts
- **Responsive Design**: 
  - Consider breakpoints: `sm`, `md`, `lg`, `xl`, and `2xl` for ultra-wide displays. 
  - Do not lock into every breakpoint if responsive design will sufficiently work.
  - Strive toward a whiteboard type implementation that allows access to tools and research in place.
  - Minimal scrolling
  - Drag and drop
- **Mobile**: Complexity of draft details will require a unique mobile design using the `sm` breakpoint.
